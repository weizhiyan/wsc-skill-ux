# 图片提示词模板

适用：即梦、豆包、千问等国内平台（默认）；Midjourney / SD 仅在用户明确指定时使用。
参考记忆：优先参考 `memory/image_patterns.md`。

## 输出结构

直接输出提示词，不输出分析过程（目标理解、画面核心设定等不展示给用户）。

风格、构图、光影、材质等要素直接体现在提示词内容里。

```
主提示词（代码块）

可选增强词（代码块，有必要时才输出）

可选替代方向（1~2条，用户没要求可省略）
```

## 追问重点
当信息不足时，优先追问：
- 你要什么类型：海报、KV、插画、图标、概念图，还是产品图？
- 更偏什么风格：科技、极简、未来感、国风、写实、3D、插画，还是别的？
- 主要主体、色调、比例、用途是什么？

## 输出要求
- 提示词必须放在代码块中
- 默认中文提示词，适配即梦/豆包/千问
- 用户说 MJ / Midjourney / SD 时才输出英文格式
- 不堆砌无意义形容词
