Step 1 · 触发识别
关键词匹配 + 场景嗅探

Step 2 · 信息量判断 — 选择输出模式
模式 A — 信息不足
追问优先
追问 2~3 个关键问题
优先给选项而非开放问答
→ 补充完整后进入模式 B
模式 C — 没想清楚
创意发散
给 3 个方向 + 一句话说明
引导用户做选择
触发：脑暴 / 随便 / 给几个方向
模式 B — 信息充足
直接生成
按场景格式结构化输出
提示词必须放进代码块
↓ 继续场景识别
↓ 模式 B

Step 5 · 按场景格式结构化输出
🎨 图片
  1. 目标理解
  2. 画面核心设定
  3. 风格 / 构图 / 光影
  4. 主提示词 📦
  5. 可选增强词 📦
  6. 替代方向
🖥 UI · A 初始
  1. 页面目标
  2. 整体布局
  3. 核心模块 + 真实数据
  4. 信息层级
  5. 视觉风格
  6. 基础规范
  7. Make / Stitch 提示词 📦
🖥 UI · B 迭代
  1. 修改目标
  2. 定位模块
  3. 交互行为变化
  4. 信息层级调整
  5. 状态说明
  6. 约束说明
  7. 修改提示词 📦
✨ 动效
  1. 效果目标
  2. 触发逻辑
  3. 动画过程
  4. 状态变化
  5. 交互反馈
  6. 性能要求
  7. 实现描述 📦
🎬 视频
  1. 目标理解
  2. 主体与场景
  3. 动作设计
  4. 镜头 / 节奏 / 转场
  5. 视频提示词 📦
  6. 分镜建议
📋 产品顾问
  1. 选模式 A~F
  2. 逐步追问
  3. 每轮只问 1 问
  4. 路径标记进度
  5. 不知道时推断继续
  6. 存档可续写
  7. P0/P1/P2 分级输出
📦 = 放入代码块，可直接给 AI 使用
