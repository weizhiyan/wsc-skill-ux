# wsc-ux (WSC Creative Suite)

> 专业的 UI 设计提示词生成（Make/Stitch）与交互需求深度优化工具，隶属于 **WSC Creative Suite** 系列。

将你的产品想法，转化为专业的 UI 视觉描述或可直接执行的 Make/Stitch 提示词。专注于交互细节、状态逻辑与专业设计规格。

---

## 🎨 WSC Creative Suite 系列成员

- **[wsc-image](https://github.com/weizhiyan/wsc-image)** — AI 视觉创意专家（生图、视频提示词优化）
- **[wsc-ux](https://github.com/weizhiyan/wsc-idea)** — UI/UX 设计与需求专家（界面、交互优化）

---

## 核心能力

| 能力 | 说明 |
|------|------|
| 交互需求优化 | 将口语想法转化为包含动效、触感反馈、状态切换的专业规格说明 |
| UI 提示词生成 | 针对 Make/Stitch 优化的结构化描述，涵盖布局、层级与真实数据模拟 |
| 智能弹窗引导 | 使用 `AskUserQuestion` 提供专业配色、动效曲线、场景目标的快速点选 |
| 记忆轴存储 | 记录用户偏好的视觉风格、色系习惯与特定指令集 |
